# 제이린쌤 뮤비 자동 제작기

서버비를 거의 쓰지 않도록 만든 정적 웹앱입니다. 사용자의 브라우저에서 참고 이미지, 프롬프트, 생성 이미지, 영상 렌더링을 처리하고 서버에는 파일을 저장하지 않습니다.

## 핵심 기능

- 참고 이미지 업로드
- 기본 프롬프트 입력
- 시리즈 템플릿 선택
- 시니어채널 두왑 시리즈 60개 고정 프롬프트
- 참고 이미지 스타일을 반영한 60개 장면 프롬프트 생성
- 사용자 OpenAI API 키로 GPT Image 2 이미지 자동 생성
- 브라우저에서 5분 영상 자동 렌더링
- 완성 영상 다운로드
- 배경음악 업로드
- 16:9, 9:16, 1:1 비율 선택

## 사용 방법

1. 브라우저에서 앱을 엽니다.
2. 수강생 본인의 OpenAI API 키를 입력합니다.
3. 시리즈 템플릿을 선택하거나 참고 이미지와 기본 프롬프트를 넣습니다.
4. `전체 자동 제작`을 누릅니다.
5. 프롬프트 생성, 이미지 생성, 영상 렌더링이 순서대로 진행됩니다.
6. 완성 파일이 자동 다운로드됩니다.

단계별로 확인하고 싶으면 `60개 프롬프트 만들기`, `이미지 자동 생성`, `영상 생성` 버튼을 따로 눌러도 됩니다.

## 시리즈 템플릿

- 시니어채널 두왑 시리즈: 1960년대 캘리포니아, 다이너, 클래식카, 주크박스, 해변 보드워크 60개 고정 장면
- 시니어채널 명화 시리즈: 중세/르네상스풍 아름다운 여인, 성 내부, 정원, 촛불, 유화 질감, 아주 미세한 명화 모션 스타일
- 이탈리안 러브송 시리즈: 빈티지 이탈리아 로맨스 스타일
- 여름 빈티지 시리즈: 해변, 우산, 자전거, 여름 휴가 빈티지 스타일

- 할리우드 워킹 빈티지 시리즈: 고전 할리우드 거리와 스타 패션 스타일

## 프롬프트 조립 규칙

각 시리즈의 고정 장면 프롬프트는 아래 구조로 자동 조립됩니다.

```text
[시리즈 전체 공통 프롬프트]
+
[개별 이미지 프롬프트]
+
highly detailed, cinematic photo realism, cohesive vintage styling, soft film grain, warm nostalgic texture, elegant composition, no modern objects, no contemporary fashion logos, no text, no watermark
```

영상 렌더링 모션 규칙:

```text
very slow pan, subtle zoom in, slight parallax depth, soft drifting motion, smooth crossfade transitions, elegant pacing, no sudden movement, no aggressive camera shake
```

## 운영 구조

- 웹앱 배포: Cloudflare Pages, GitHub Pages, Netlify 같은 정적 호스팅 무료 플랜 사용 가능
- 이미지/영상 생성: 사용자가 입력한 Google API 키로 실행
- 영상 조립: 사용자 브라우저에서 처리
- 저장 방식: 서버 저장 없음

## 로컬 실행

```powershell
cd C:\Users\user\my-projects\ai-long-video-maker
python -m http.server 4173 --bind 127.0.0.1
```

브라우저에서 엽니다.

```text
http://127.0.0.1:4173
```

## 배포

`ai-long-video-maker` 폴더 전체를 Cloudflare Pages 또는 GitHub Pages에 올리면 됩니다.

## 주의

- 선생님 API 키를 수강생에게 공유하지 마세요.
- 5분 영상 생성은 브라우저 실시간 녹화 방식이라 실제로 약 5분이 걸립니다.
- 브라우저에 따라 결과 형식이 `mp4`가 아니라 `webm`일 수 있습니다.
- 사용자의 컴퓨터 성능이 낮으면 60장 이미지 기반 렌더링이 느릴 수 있습니다.
- 기본 이미지 모델은 `gpt-image-2`입니다. 비용을 아끼려면 장면 수와 이미지 품질을 낮춰 먼저 테스트하세요.
